# 🚀 Cryptocurrency Price Tracker

A modern and responsive Cryptocurrency Price Tracker website that displays real-time prices of popular cryptocurrencies such as Bitcoin, Ethereum, and Dogecoin using the CoinGecko API.

This project is built using HTML, CSS, and JavaScript to demonstrate API integration, asynchronous data fetching, and responsive web design.

## 🌟 Features

- 📈 Live cryptocurrency prices
- 💰 Bitcoin, Ethereum, and Dogecoin price tracking
- 🔄 Real-time API integration using CoinGecko
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🎨 Modern user interface
